# ABU ROBOCON CAD Design

## Overview

This repository contains the CAD design and assembly of our ABU ROBOCON robot developed using SolidWorks.

The project focuses on creating a competition-ready robot with a lightweight structure, modular subsystems, and manufacturable components.

## Software Used

- SolidWorks 2025
- SolidWorks Simulation

## Responsibilities

My contributions included:

- Chassis design
- Assembly creation
- Mechanism integration
- Manufacturing drawings
- Design optimization
- Motion Study/Simulation

## Key Features

- Modular architecture
- Lightweight frame
- Easy maintenance access
- Competition-oriented design

## Skills Demonstrated

- Part Modeling
- Assembly Design
- Motion Analysis
- Design for Manufacturing
- Technical Drawings
